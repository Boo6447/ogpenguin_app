CP Atake App</h1>

> Desktop client for cpatake.dink.cf


### For Developers
You need [node.js](https://nodejs.org). The latest version is recommended but others work too.

Installation:
```sh
git clone https://github.com/frostygg/desktop
cd desktop
npm i
npm start
```
Building:
```sh
npm run make-win
npm run make-mac
npm run make-linux
```
